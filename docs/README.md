# OctoAcme Project Management Documentation

## Overview
OctoAcme follows a structured, iterative project management approach that emphasizes customer value, clear ownership, data-driven decisions, and psychological safety. This documentation suite provides guidance for all project lifecycle phases and links to phase-specific process documents.

## Core Principles
- Customer-first: Prioritize customer value and usability
- Iterative delivery: Deliver small, testable increments
- Clear ownership: Each project has named leads
- Data-informed: Measure impact and iterate
- Psychological safety: Encourage feedback and learning

## Quick Start for New Team Members
1. Read the Project Management Overview to understand principles and roles
2. When starting a new project, follow the Project Initiation Guide
3. Use the Project Planning and Execution guides as your project progresses
4. Refer to Roles & Personas for responsibilities and handoffs

## How the documents connect in the lifecycle
Initiation → Planning → Execution → Release → Retrospective
